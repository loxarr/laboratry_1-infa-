Я попробовал написание touch script.bash и gedit script.bash, где Touch — команда Unix, которая используется для создания пустых файлов и gedit - запуск текстого редактора в файлк script.bash&
В файле я написал #!/bin/bash( для указание, где хранится bash) и echo "Welcome to ITMO University". Echo — это встроенная команда в Linux, которая выводит строку текста в терминал.
